# weder

Command-line weather using the Forecast API

## Example

Get the current conditions by zip code:

```shell script
weder --zip=12345 --apiKey=abcdef12345
```

## Requiremens for Building
- C++20
- [CMake](https://cmake.org)
- [Conan](https://conan.io)
